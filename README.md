# KingChords II 🎹

En HiChord-inspirerad ackord-synth i webbläsaren, stylad som en Teenage Engineering EP-133.
Allt ligger i en enda `index.html` (Web Audio API + [smplr](https://github.com/danigb/smplr)) — inget byggsteg.

## Funktioner

- **7 ackord-pads** med de diatoniska ackorden, i 3 layouter (kvintcirkel / funktion / skala)
- **Tonalitets-pad** (sus2 · sus4 · add9 · 6 · 7 · maj7) som morfar ackordet live
- **Bas-pad** med omvändningar + **auto voice-leading** (väljer röstläggning med minsta tonrörelse)
- **Instrument:** bas, trummor (TR-808), piano, orgel, gitarr — riktiga samplade ljud
- **Scratch-skiva** (vinyl) med tre ljud att scratcha
- Byt tonart (grundton + dur/moll), oktav, filter och volym

## Köra lokalt

Öppna bara `index.html` i en webbläsare. Instrumentljuden hämtas från ett ljudbibliotek på nätet första gången (kräver internet).

## Spela

Tryck på paderna eller tangenterna `1`–`7`. Tonalitet: `Q`–`U`. Bas: `A`/`S`/`D`/`F`. Oktav: `Z`/`X`.
